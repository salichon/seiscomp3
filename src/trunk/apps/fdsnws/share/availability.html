SeisComP3 Data Availability Web Service

The data availability web service returns detailed time span information of what timeseries data is available at the DMC archive.

Available URLs

  • extent - Produces list of available time extents (earliest to latest) for selected channels (network, station, location and quality) and time ranges.
  • builder-extent - URL builder helping you to form your data extent requests
  • query - Produces list of contiguous time spans for selected channels (network, station, location, channel and quality) and time ranges.
  • builder - URL builder helping you to form your data time span requests
  • version

Interface Documentation

The availability service is no official standard yet. This implementation aims to be compatible with the IRIS DMC IRISWS availability Web Service implementation.

Feature Notes

  • back-end software: SeisComP3
  • The IRISWS availability implementation truncates the timespans of the returned data extents and segments to the requested start and end times (if any). This implementation truncates the start and end time only for the formats: sync and request. The text, geocsv and json format will return the exact time windows extracted from the waveform archive.
